# FastAPI Database API

This FastAPI project demonstrates full CRUD operations with:

- SQLAlchemy ORM
- SQLite database
- Pydantic models
- Dependency injection
- Swagger docs

## Features
- Create tasks
- List all tasks
- Retrieve a task by ID

## How to Run
uvicorn main:app --reload
