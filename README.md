# 🚀 FastAPI CRUD API

## 📌 Overview

This project is a RESTful API built using FastAPI that performs CRUD operations. It allows users to create, read, update, and delete data efficiently.

---

## ⚙️ Tech Stack

* Python
* FastAPI
* MongoDB
* Postman

---

## 🔥 Features

* Create, Read, Update, Delete operations
* REST API architecture
* Fast performance using FastAPI
* JSON-based data handling
